Service Desk System (Pick IT Up)

In this project my partner and I were asked to program a service desk system. This involved the use of Java and GitHub. The system included user account creation with secure login and logout functionality.

It provided a visual interface to view tickets from a Derby database, along with filtering and status updates (Open, Closed, In-Review). The platform also supported different account types, including IT staff and Guest users.

Java Derby Database GitHub GUI Development

Campbell and Starr Bakers' Network

Based on a case study, my team and I were appointed to design and implement a robust network that meets the requirements of the business owners. In this project, I configured network devices of the virtual network, including the servers and the PC hosts.

Together we installed the Active Directory, the DNS, and DHCP server. We also made IP configurations and wrote a PowerShell script to automate the creation of new AD users and assign their permissions.
